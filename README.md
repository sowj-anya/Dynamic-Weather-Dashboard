# 🌦️ Dynamic Weather Dashboard

## 📌 Overview
Dynamic Weather Dashboard is a simple web application that provides real-time, location-based weather information.  
It helps users check temperature, weather conditions, and forecasts for better daily planning and travel decisions.

## 🚀 Features
- 🌍 Automatic location detection using Geolocation API  
- 🌡️ Real-time temperature display  
- ☁️ Weather conditions (clear, rain, clouds, etc.)  
- 📅 Basic weather forecast  
- 💻 Clean and responsive user interface  

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **APIs:** OpenWeatherMap API, Geolocation API  

## ⚙️ How It Works
1. User allows location access  
2. Application captures latitude and longitude  
3. Sends request to OpenWeatherMap API  
4. Receives weather data  
5. Displays the data on the dashboard
